Learning Flexbox & Grid

A reflection on designing with layout systems

In this project, I explored how Flexbox and Grid can be used to build responsive layouts. At first, it was confusing to understand how elements align and respond to different screen sizes.

Flexbox was helpful for one-dimensional layouts, especially for aligning items in rows. Grid, on the other hand, made it easier to structure full page sections.

What I learned

  • How Flexbox controls alignment and spacing
  • How Grid creates structured layouts
  • How reusable components improve design consistency

Overall, this assignment helped me understand how modern websites are built using modular design systems instead of single static layouts.
